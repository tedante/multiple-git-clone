# Multiple Git Clone

## 1. Adjust file:
> 1. organization.txt
> 2. prefix.txt
> 3. usernameList.txt

## 2. Run command:
```bash
bash ./multiple-clone.sh <org_name> <prefix>

# example
bash ./multiple-clone.sh h8-fsjs-gitclass-p1-23 fsjs-p1-v1-c1
```
